# Mi Portafolio (Bootstrap + componentes interactivos)

Este proyecto consiste en una versión actualizada de mi portafolio personal, maquetada con **Bootstrap 5** usando **CDN** para mejorar la experiencia visual y la responsividad (mobile y desktop). El objetivo es presentar mi información y facilitar el contacto mediante componentes de Bootstrap y estilos propios.

## Características
- **Navbar de Bootstrap responsiva** en **todos los archivos**, con menú colapsable en mobile.
- **Galería en `index.html` con Carousel** para mostrar contenido visual de forma interactiva.
- Página adicional en `pages/` con contenido maquetado con componentes de Bootstrap (por ejemplo **cards**), más **estilos personalizados** en CSS para ajustar la estética.

## Tecnologías
- HTML5
- CSS3 (grid/flexbox y estilos personalizados)
- Bootstrap 5 (CDN)

## Estructura del proyecto
- `index.html` (home / presentación principal con carousel y secciones maquetadas)
- `pages/` (página adicional con componentes de Bootstrap)
- `styles.css` (estilos personalizados sobre Bootstrap, si aplica)
- `assets/` (imágenes y recursos utilizados en la interfaz)

## Bootstrap implementado
- **Navbar** (responsiva)
- **Carousel** (galería interactiva)
- **Cards** / componentes de contenido en la página adicional

## Cómo correr el proyecto
Abrir `index.html` en el navegador (o desplegar el repositorio si se realiza con GitHub Pages / hosting).
