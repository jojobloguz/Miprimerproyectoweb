# Mi Portafolio (Bootstrap + componentes interactivos)

Este proyecto es una actualización de mi portafolio integrando **Bootstrap vía CDN** para mejorar la experiencia con componentes interactivos.

## Características
- Navbar de Bootstrap **responsiva** (incluye menú hamburguesa en móvil) en **todos los archivos**
- Sección de **Galería con Carousel** en `index.html`
- Página adicional en `pages/` con componentes de Bootstrap (y estilo propio con CSS)

## Tecnologías
- HTML5
- CSS3 (incluye Grid/Flexbox y estilos personalizados)
- Bootstrap 5 (CDN)

## Estructura del proyecto
- `index.html`
- `pages/` (página adicional elegida)
- `styles.css` (si aplica)
- `assets/` (imágenes u otros recursos)
